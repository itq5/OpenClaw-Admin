## 5. 部署信息

### 5.1 部署方式
**推荐方式**: Docker Compose 部署
```bash
cd /www/wwwroot/ai-work/
docker-compose pull
docker-compose up -d
```

**备选方式**: PM2 部署
```bash
npm ci --production
npm run build
pm2 start ecosystem.config.js
```

### 5.2 健康检查
```bash
# 服务状态检查
curl http://localhost:3000/api/health

# PM2 状态
pm2 list
```

### 5.3 回滚方案
```bash
# 代码回滚
git checkout v1.3.0
git checkout -b rollback-v1.3.0

# 数据库回滚 (如有需要)
mysql -u root -p ai_work < backup.sql
```
