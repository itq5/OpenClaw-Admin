#### ❌ 失败的测试 (24 个用例)

**失败原因分类**:

1. **Vue I18n 配置问题** (19 个失败)
   - 组件直接使用 `useI18n()` 但未在测试环境中安装插件
   - 受影响组件:
     - `BatchActionsBar.vue` (7 个失败)
     - `CronEditor.vue` (14 个失败)
     - `DashboardCard.vue` (6 个失败)
     - `SmartSearchFilter.vue` (9 个失败)
     - `ThemeSwitcher.vue` (8 个失败)
   
   **错误信息**: `SyntaxError: Need to install with \`app.use\` function`

2. **Cron Store 测试问题** (11 个失败)
   - Mock API 调用未正确设置
   - selectedJobId 为 null 导致赋值失败
   
   **错误示例**:
   ```
   TypeError: Cannot set properties of null (setting 'value')
   at tests/unit/cron-store.test.ts:219:27
   ```

3. **TypeScript 编译超时** (1 个失败)
   - `code-quality.test.ts` - TypeScript 编译测试超时 (5000ms)
