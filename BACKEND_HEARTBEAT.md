## 接口实现详情

### 1. 批量删除接口
```
DELETE /api/v1/batch/:resource
```
- 支持资源：users, tasks, scenarios, audit-logs
- 权限：{resource}:delete
- 特性：SQL 注入防护、操作日志记录

### 2. 批量状态变更接口
```
PATCH /api/v1/batch/:resource/status
```
- 支持资源：users, tasks, scenarios, audit-logs
- 权限：{resource}:update
- 特性：批量更新、受影响行数统计

### 3. 批量查询接口 (新增)
```
POST /api/v1/batch/:resource/batch-get
```
- 支持资源：users, tasks, scenarios, audit-logs
- 权限：{resource}:read
- 特性：
  - 支持指定返回字段（字段白名单）
  - 防止 SQL 注入
  - 返回数据计数

### 4. 批量导出接口
```
POST /api/v1/batch/:resource/export
```
- 支持资源：users, tasks, scenarios, audit-logs
- 权限：{resource}:read
- 特性：CSV 格式导出、字段选择

### 5. 批量分配接口
```
PATCH /api/v1/batch/:resource/assign
```
- 支持资源：仅 tasks
- 权限：tasks:update
- 特性：批量任务分配、分配计数

## 安全特性

1. **SQL 注入防护**: 所有接口使用参数化查询
2. **XSS 防护**: 输入数据经过 WAF 过滤
3. **权限验证**: JWT 认证 + RBAC 权限验证
4. **字段白名单**: 批量查询接口使用字段白名单机制
5. **操作审计**: 所有批量操作记录到审计日志
